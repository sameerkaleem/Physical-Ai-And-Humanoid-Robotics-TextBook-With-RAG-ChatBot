# فزیکل ای آئی اور ہیومنوائڈ روبوٹکس میں خوش آمدید

یہ ٹیکسٹ بک فزیکل ای آئی اور ہیومنوائڈ روبوٹکس کی دلچسپ دنیا کا جائزہ لیتی ہے، ذہانت کو جسمانی نظام کی ایک Emergent خصوصیت کے طور پر سمجھتے ہوئے۔

## زیر بحث موضوعات

- سینسنگ اور ایکٹوایشن
- کنٹرول سسٹم
- روبوٹکس میں مشین لرننگ
- جسمانی ذہانت
- ہیومنوائڈ ڈیزائن کے اصول

ہر موضوع میں گہرائی سے جانے کے لیے نیویگیشن سائڈ بار میں مختلف ابواب کو دیکھیں۔
